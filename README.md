# DevOps Project: Jenkins + Git + Docker + Email Pipeline

This project demonstrates a complete CI/CD pipeline using Jenkins, Git, Docker,
and Email notifications.

## Contents
- app/ → Node.js Application
- Dockerfile → Build container image
- Jenkinsfile → CI/CD pipeline
- REPORT.md → Full documentation for submission

## How to Use
1. Upload project to GitHub
2. Update Jenkinsfile with your repo URL + Docker Hub username
3. Add Jenkins credentials:
    - dockerhub-username
    - dockerhub-password
4. Run Jenkins pipeline
